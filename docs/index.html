Unit fractions

by Thomas F. Bloom and Bhavik Mehta

Blueprint GitHub

What is it about?

This project contains a formalized proof of the main result of the preprint ‘On a density conjecture about unit fractions’ using the Lean theorem prover, mainly developed at Microsoft Research by Leonardo de Moura. This project structure is adapted from the infrastructure created by Patrick Massot for the Sphere Eversion project.

More precisely we will prove that, any dense set of integers contains distinct integers whose reciprocals sum to 1. (This generalises a colouring result of Croot, and resolves an old open problem of Erdős and Graham.) For further details, precise statements, context, and so on, we refer to the preprint itself.
